Class Server cодержит следующие методы:
public void start(String QMName, String Q1_name, String Q2_name) – метод устанавливает соединение с БД, с менеджером очередей, открывает очереди на запросов и ответов на чтение и запись соответственно, подсчитывает количество обработанных запросов. На вход принимает имя администратора очередей и названия 2х очередей.
private boolean processQuery() – метод проверяет код запроса и другие параметры из очереди, взаимодействует с БД и отправляет ответы назад клиенту (так же через очередь). Метод возвращает true в случае успеха и false в обратном случае.


Class DBServer cодержит следующие методы:

public SQL(String DBName, String ip, int port) – конструктор класса. Подключается к БД.
public int addCat(int id, String name) – Добавление новой категории. Принимает на вход id новой категории и ее название. Формирует и отправляет запрос к базе. Возвращает 1 при успешном выполнении, в противном случае возвращает 0.
public int addNews(int id, String name, int id_cat) - Добавление новой новости. Принимает на вход id новой новости ее название и id категории в которой будет находится эта новость. Формирует и отправляет запрос к базе. Возвращает 1 при успешном выполнении, в противном случае возвращает 0.
public ArrayList<NWS> showCat() – Вывод всех категорий. Формирует и отправляет запрос к базе. Возвращает объект класса ArrayList<NWS>  при успешном выполнении, в противном случае возвращает null.
public ArrayList<NWS> showNews() - Вывод всех новостей. Формирует и отправляет запрос к базе. Возвращает объект класса ArrayList<NWS>  при успешном выполнении, в противном случае возвращает null.
public int deleteNews(int id) – Удаление новости. Формирует и отправляет запрос к базе. Возвращает 1 при успешном выполнении, в противном случае возвращает 0.
